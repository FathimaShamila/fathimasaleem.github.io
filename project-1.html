RAG Powered Q&A Chatbot

An AI powered document Intelligence system that lets users upload documents and ask context aware questions with semantic search and source attribution.

Project Overview

The RAG Chatbot is an AI powered document Intelligence & Q&A system built with Python,LlamaIndex and Gradio and the Google Gemini API. It allows users to upload multi page PDFS ,which are first parsed using PyMuPDF and then split into sentence based chunks with overlap to preserve context across sections. Each chunk is wrapped in a Document object from LlamaIndex and then converted into embeddings using the Sentence Transformers embedding model and the resulting embeddings are indexed and stored in a FAISS vector store for efficient semantic retrieval. The chatbot features a user friendly Gradio interface, with a status box indicating when documents are processed and a document analysis panel displaying statistics about the processed documents.

Users can ask context aware questions, and the chatbot provides accurate answers along with the source of the information, highlighting the most relevant chunk. Out of context queries are handled gracefully with clear message, helping prevent hallucinations by ensuring the model only responds when sufficient information is available.This project demonstrates the full RAG workflow, including document parsing, syntactic chunking with overlap, embedding generation, semantic retrieval with FAISS, multi document handling, and retrieval augmented generation , all combined in an interactive and accessible frontend interface.
